# 🎨 Cornell Box — Ray Tracing (WinForms)

Реализация рендеринга классической сцены **Cornell Box** методом трассировки лучей (Ray Tracing) на C# с визуализацией через Windows Forms.

---

## 📌 Описание

**Cornell Box** — эталонная тестовая сцена в компьютерной графике, предложенная исследователями Корнеллского университета в 1984 году. Она используется для оценки точности алгоритмов глобального освещения путём сравнения рендера с фотографией реальной физической модели.

Данный проект — учебная реализация (Индивидуальное задание №2), демонстрирующая:
- Построение 3D-сцены программным способом (без сторонних 3D-движков).
- Алгоритм трассировки лучей (Ray Tracing) с поддержкой отражений, теней и диффузного освещения.
- Вывод результата в окне Windows Forms через растровое изображение (`Bitmap`).

---

## 🚀 Возможности

- 🔦 **Источник света** — точечный / площадной источник на потолке сцены.
- 🟥🟩 **Цветные стены** — классическая конфигурация: красная (левая), зелёная (правая), белые (пол, потолок, задняя).
- 🧊 **Геометрические объекты** — параллелепипеды / сферы внутри сцены.
- 🌗 **Модель освещения** — диффузное (Ламбертово) освещение, тени, опционально зеркальные отражения.
- 🖼️ **Рендеринг в Bitmap** — попиксельный расчёт цвета с выводом на `PictureBox`.
